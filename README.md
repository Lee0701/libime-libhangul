# libime-libhangul
libIME2와 libhangul을 이용한 간단한 윈도우용 한국어 입력기입니다.

## 주요 기능
* 다양한 한글 자판 지원 (3beol판 커스텀 libhangul 사용)
* 실시간 자동 한자 변환 지원

## 설치 및 설정 방법
설치 마법사는 현재 준비 중에 있습니다. 사용해 보시려면 dll을 수동으로 등록해야 합니다. 프로젝트 루트의 install.cmd와 uninstall.cmd를 관리자 권한으로 실행해서 dll을 등록/해제할 수 있습니다.

uninstall.cmd를 실행하기 전에는 이 입력기를 사용했던 프로그램을 모두 종료해 주세요. 윈도우 재부팅 또는 재로그인 후 이 입력기를 사용하지 않은 상태에서 진행하면 확실하게 제거됩니다.

설치 후 자세한 설정 방법은 doc/settings.md 및 doc/keyboards.md 를 참고하세요.
